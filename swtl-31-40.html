Date() create a new Date object to get the current date and time

Chapter 32 — Extracting parts of the date and time

The Date() object has methods to get different time format.

getDay() - Get the day from the Date object.

getMonth() - Get the month from the Date object.

getDate() - Get the date from the Date object.

getFullYear() - Get the year from the Date object.

getHours() - Get the hour from the Date object.

getMinutes() - Get the minute from the Date object.

getSeconds() - Get the second from the Date object.

getMillisecond() - Get the millisecond from the Date object.

getTime() - Get milliseconds since midnight, January 1, 1970.

Chapter 34 — Changing elements of a date and time

Using the set methods to change parts of a Date object.

setFullYear() - Set the year to the Date object.

setMonth() - Set the month to the Date object.

setDate() - Set the date to the Date object.

setHours() - Set the hour to the Date object.

setMinutes() - Set the minute to the Date object.

setSeconds() - Set the second to the Date object.

setMilliseconds() - Set milliseconds past the second of the Date object.

getMillisecond() - Get the millisecond from the Date object.

getTime() - Get milliseconds since midnight, January 1, 1970.

Chapter 35 — Functions

A function is a block of code that does some specific task and can be reused when that task is needed to be done again.

Chapter 36 — Functions: passing them data

A function can be set up to allow users to give it some data when it is called.

Chapter 37 — Functions: passing data back from them

A function can return a value to the code that calls it.

Chapter 38 — Functions: local vs. global variables

A variable declared inside the main code is a global variable and known globally by all functions in that code.

A variable declared inside a function is a local variable and is known only inside that function.

Chapter 39 starts teaching about the switch statements and Chapter 40 teaches how to complete the statements.
