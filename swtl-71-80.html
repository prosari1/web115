Chapter 75 — Objects: Checking for properties and methods

The hasOwnProperty() method returns a boolean indicating whether the object has the specified property as its own property (as opposed to inheriting it).

Chapter 76 — Browser control: Getting and setting the URL

window.location.href detects the current URL and assigns it to a variable that hasn't been declared yet.

Chapter 77 — Browser control: Getting and setting the URL another way

The assign() method loads a new document.

The replace() method replaces the current document with a new one.

The reload() method is used to reload the current document. It does the same as the reload button in your browser.

Chapter 78 — Browser control: Forward and reverse

The back() method loads the previous URL in the history list. This is the same as clicking the "Back button" in your browser.
