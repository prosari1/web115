Chapter 41 — while loops

The while loop, like the for loop, loops through a block of code as long as a specified condition is true.

  • var counter is initialized with a value before the loop.
  • The counter is updated within the code.

Chapter 42 — do...while loops

The do/while statement creates a loop that executes a block of code once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

Chapter 43 — Placing scripts

In HTML, JavaScript code is inserted between <script> and </script> tags.

You can place any number of scripts in an HTML document. Scripts can be placed in the <body>, or in the <head> section of an HTML page, or in both.

Chapter 44 — Commenting

Comments can be used to explain JavaScript code, and to make it more readable.

Single line comments start with //

Multi-line comments start with /* and end with */.
