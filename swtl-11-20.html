Smarter Way to Learn Chapters 11-20

Chapter 11 — Comparison operators

These are the comparison operators: ===, !==, >, and <. You can compare different data types using the comparison operators.

Chapter 12 — if...else and else if statements

Else test for a condition when all the previous tests have failed.


Chapter 13 — Testing sets of conditions

To test more than one condition using the if statement you can add (&&) two conditions; or use the || to test if either condition is true.


Chapter 14 — if statements nested

An alternative to using && to test multiple conditions in a single if statement is to use nested ifs statements. The second if statement needs to be indented.

Chapter 15 — Arrays

An array comprises any number of values. In an array, the numbering starts with zero.


Chapter 16 — Arrays: adding and removing elements

POP removes the last element from an array.

PUSH adds one or more elements to the end of an array.

Chapter 17 — Arrays: removing, inserting, and extracting elements

SHIFT removes first element of an array.

UNSHIFT adds one or more elements to the beginning of an array

SPLICE inserts new elements anywhere in an array and/or removes elements from an array

SLICE copies elements from an array to another array.

Chapter 19 — for loops: flags, Booleans, array length, and loopus interruptus

Chapter 20 — for loops nested

For loops can be nested inside another. In this case, so if you have 3 iterations in one loop and the inner loop with 4 iterations, you're going to have a total of 12 iterations. The loop stops when all iterations are complete.
